Welcome to Blank of the Day

Simple Project Using the Twitter API and Google Custom Search API

How Does It Work?

The backend is set to monitor specific #hashtags. Once per day, the twitter search API is used to collect recent tweets that contain that hashtag. The most common tweet is then used to create a google image search query.

The query is used to do a google image search and an image is randomly chosen from the results. The tweet and the image are used to create a post showing the [blank] of the day. Check out the repo for more information.
